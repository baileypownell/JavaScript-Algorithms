Given an array of integers, you are allowed to change each element by increases/decreases of one such that the array's integers are always increasing by one value from one element to the next. Write a function that finds the minimal number of value shifts/upticks required to obtain a strictly increasing sequence from the input (so 4,5,6,7 rather than something like 4,6,8,10, which while increasing, is increasing each time by more than ONE value shift.
